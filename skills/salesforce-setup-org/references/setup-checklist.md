# Scratch Org Setup Checklist

## 事前確認

- scratch org definition file の場所
- Dev Hub alias
- default alias の命名
- org を作り直してよいか

## 初期デプロイ

- package dir ごとの deploy 順序
- unpackaged metadata の deploy 有無
- 初期ユーザー作成や Permission Set 割当の有無

## 手動作業

- FlexiPage の Activate
- Experience Cloud や承認プロセスなど手動設定
- 別ユーザーでのログイン確認

## 完了判定

- 主要 package が deploy 済み
- 必須ユーザーと権限が揃っている
- seed 実行へ進める状態になっている
